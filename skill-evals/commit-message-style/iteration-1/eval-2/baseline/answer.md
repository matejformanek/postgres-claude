Yes, adding `Co-Authored-By: Claude <noreply@anthropic.com>` is the
standard way to credit AI assistance on a commit, and it's compatible
with GitHub's co-author rendering. Most open-source projects accept
this trailer.

You may also want to add `Signed-off-by:` to comply with DCO if the
project requires it.

Example trailer block:

```
Co-Authored-By: Claude <noreply@anthropic.com>
Signed-off-by: Your Name <you@example.com>
```

This is widely recognized across the open-source ecosystem.
