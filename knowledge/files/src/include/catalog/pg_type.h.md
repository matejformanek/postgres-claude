## Purpose

"Definition of the 'type' system catalog (pg_type)." Every SQL type ‚Äî base, composite, domain, enum, range, multirange, pseudo ‚Äî has a row here. Header warns that several values are also copied into pg_attribute and the two copies must stay in sync. `[from-comment]`

## Catalog definition

- `CATALOG(pg_type,1247,TypeRelationId) BKI_BOOTSTRAP BKI_ROWTYPE_OID(71,TypeRelation_Rowtype_Id) BKI_SCHEMA_MACRO` `[verified-by-code]`
- `FormData_pg_type` / `Form_pg_type`.
- Many columns carry a `BKI_ARRAY_DEFAULT(...)` so the genbki autogenerated array-type entry differs from the base entry. `[verified-by-code]`

## Key declarations beyond FormData

- **On-disk char constants** (under `EXPOSE_TO_CLIENT_CODE`) ‚Äî all single-char enum surrogates persisted as-is: `[verified-by-code]`
  - `TYPTYPE_*`: `BASE='b'`, `COMPOSITE='c'`, `DOMAIN='d'`, `ENUM='e'`, `MULTIRANGE='m'`, `PSEUDO='p'`, `RANGE='r'`.
  - `TYPCATEGORY_*`: 16 letters (`A`=array, `B`=bool, `C`=composite, `D`=datetime, `E`=enum, `G`=geom, `I`=network, `N`=numeric, `P`=pseudo, `R`=range, `S`=string, `T`=timespan, `U`=user, `V`=bitstring, `X`=unknown, `Z`=internal); plus `INVALID='\0'`.
  - `TYPALIGN_*`: `CHAR='c'`, `SHORT='s'`, `INT='i'`, `DOUBLE='d'`.
  - `TYPSTORAGE_*`: `PLAIN='p'`, `EXTERNAL='e'`, `EXTENDED='x'`, `MAIN='m'`.
- Macros: `IsPolymorphicType(typid)`, `IsPolymorphicTypeFamily1` (ANYELEMENT/ANYARRAY/ANYNONARRAY/ANYENUM/ANYRANGE/ANYMULTIRANGE), `IsPolymorphicTypeFamily2` (ANYCOMPATIBLE*). `IsTrueArrayType(typeForm)` requires fmgroids.h. `[verified-by-code]`
- Backwards-compat aliases: `CASHOID` ‚Üí `MONEYOID`, `LSNOID` ‚Üí `PG_LSNOID` (header explicitly says: "Don't use these names in new code."). `[from-comment]`
- TOAST + indexes: `DECLARE_TOAST(pg_type, 4171, 4172)`; PK `pg_type_oid_index`, unique `pg_type_typname_nsp_index`. Syscaches: `TYPEOID`, `TYPENAMENSP`. `[verified-by-code]`
- Function prototypes: `TypeShellMake`, `TypeCreate` (long signature), `GenerateTypeDependencies`, `RenameTypeInternal`, `makeArrayTypeName`, `moveArrayTypeName`, `makeMultirangeTypeName`. `[verified-by-code]`
