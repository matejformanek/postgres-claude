## Discovered (out of scope — for a future pass, not fixed here)

- **The "Synthesized over" glob heuristic is unsafe and should be retired
  from the Pass 1 recipe.** It over-matches directory paths named in a note's
  prose (see Pass 1 note above). The precise `source/PATH`-cite mechanism is
  sufficient and defensible. If a future run wants the glob's coverage, it
  should parse the *explicit brace-expanded file list*
  (`nbt{ree,xlog}.h.md` → `nbtree.h.md`, `nbtxlog.h.md`) rather than globbing
  the parent directory.
- **Raw-frequency glossary ranking is now noise-dominated.** The top
  undefined tokens are source-file/module basenames and build metadata, not
  concepts; the ranker needs the basename + filename-context filters applied
  here to surface real internals terms. Consider baking those filters into
  the recipe so future runs don't re-derive them.
- **~57% of subsystem issue registers have diverged from the template.**
  The mechanical mirror can only serve the ~half still on-template. If the
  bespoke format is now the intended house style, the recipe's Pass 3 should
  say so explicitly (and the mirror's job shrinks to "seed a register that
  doesn't exist yet").
