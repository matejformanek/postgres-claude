## Tooling note (for maintainers)

This session's GitHub egress is proxy-scoped to `matejformanek/postgres-claude`;
`api.github.com/repos/postgres/postgres/*` and `github.com/postgres/postgres/*`
both return **403** regardless of `GH_TOKEN`. `raw.githubusercontent.com` and
`buildfarm.postgresql.org` are reachable. The recipe's Inputs section names
`api.github.com` + `github.com/...commit/<sha>.diff` as sources — those are dead
in the cloud sandbox. **`git.postgresql.org` gitweb (`a=atom` for the commit
list, `a=commitdiff_plain;h=<sha>` for diffs) is the working substitute** and
should be added to the recipe's Inputs/failure-modes as the primary path. Feed
caps at 20 entries (~2 days) — fine for a daily 24h window, but a multi-day
backfill would need `commitdiff_plain` walked by parent SHA instead.
